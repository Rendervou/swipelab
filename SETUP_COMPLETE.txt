🚀 QUICK START

1️⃣ Setup Database:
   php artisan migrate
   php artisan db:seed

2️⃣ Setup Storage:
   php artisan storage:link

3️⃣ Build Assets:
   npm run dev

4️⃣ Start Servers:
   Terminal 1: php artisan serve
   Terminal 2: npm run dev

5️⃣ Access App:
   http://localhost:8000

⏱️  WAKTU SETUP: ±15-30 menit

🎮 GAMIFICATION SYSTEM

Point System:
  • Upload Design: +10 points
  • Receive Like: +2 points
  • Give Feedback: +1 point

Levels (6 Total):
  • Beginner (0 pts) → Getting Started Badge
  • Intermediate (50 pts) → Contributor Badge
  • Advanced (100 pts) → Popular Creator Badge
  • Master (200 pts) → Master Creator Badge
  • Expert (500 pts) → Community Star Badge
  • Legend (1000+ pts) → Design Legend Badge

Default Badges (Auto-awarded):
  ✅ Getting Started
  ✅ Contributor
  ✅ Popular Creator
  ✅ Master Creator
  ✅ Community Star
  ✅ Design Legend
